Install
-------
.. code-block:: bash

    pip install git+git://github.com/qevo/py_data_helper.git

.. note:: Retrieve a copy of the source from `the GitHub repo <https://github.com/qevo/py_data_helper>`_.

Features
--------
Hashing support
```````````````
* Get values for md5, sha1, sha224, sha256, sha384, sha512.

Regular Expressions
```````````````````
* Wildcard patterns

Processing
``````````
* Filter a list of strings.
* Split a string by length.
